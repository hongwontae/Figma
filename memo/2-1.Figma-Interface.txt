1. Figma 하이어리키
1-1. 계층과 위계를 의미합니다.
1-2. 팀 내부에 프로젝트 내부에 파일 계층

1-3. 파일 왼쪽은 레이어, 오른쪽은 속성, 하단에는 툴바, 가운데 영역은 캔버스라고 합니다.
=> 피그마에서 디자인할 수 있는 영역을 프레임이라고 합니다.
=> 프레임 내부에 들어가는 요소들을 엘리멘트라고 합니다.
=> 유사한 프레임을 묶어줄 수 있는 데 이를 섹션이라고 합니다.
=> 섹션도 여러 개로 묶을 수 있습니다.


1-4. page => 하나의 캔버스입니다.


** 하나의 페이지를 생성하면 하나의 캔버스 영역이 생성됩니다. 캔버스 내부에는 섹션이 존재하고 섹션 안에는
프레임이 존재하고 프레임 내부에는 엘리멘트가 존재합니다.
(섹션끼리 묶여서 또 다른 섹션이 될 수 있고 프레임끼리 묶여서 또 다른 프레임이 될 수 있습니다.)
(Page > Canvas > Section > Frame > Element)



2. 피그마에서 작업하는 기본 순서 (그림 or 문서)
2-1. 도구 선택 
2-2. 그리기
2-3. 속성 설정


3. Tools (하단에 존재하는 바)
3-1. Move and Scale tools
=> v를 누르면 요소를 이동시키거나 크기를 조절할 수 있지만 일부 엘리멘트(글자)는 비율에 맞춰 커지지 않습니다.
=> k를 누르면 요소를 이동시키거나 크기를 조절하고 비율에 맞춰 커지게 합니다.

3-2. Frame and Slice tools
=> f : frame 설정
=> shift+s : 영역 설정

3-3. Creation Tool
=> p / shift + p

3-4. Text Tool
=> T : 글씨

3-5. Hand Tool
=> h : 이동할 때 사용하는 툴입니다.

3-6. Comment Tool
=> c : 원하는 곳에 주석을 달 수 있게 해줍니다.
