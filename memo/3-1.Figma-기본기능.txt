1. 피그마 단축키 전체 보기
=> Main Menu 탭에서 Help and Account에서 keyboard shortcuts를 클릭하면 전체 단축키를 볼 수 있습니다.
=> ctrl shift ?


2. 프레임 다루기
2-1. F를 클릭하여 사용하거나 하단 메뉴에 프레임을 선택하여 드래그하여 사용합니다.
2-2. Frame 모드가 적용되면 오른쪽 패널에 여러 기기에 맞는 프레임이 존재합니다.
2-3. 프레임 박스를 선택하면 모서리에 바인딩 박스가 존재합니다. => 이를 통해서 사이즈 조절이 가능합니다.
2-4. 왼쪽 상단에 프레임 이름을 설정할 수 있습니다.

2-5. 프레임 복사
1. Alt를 누르면서 드래그 하여 복사
2. ctrl + d
=> ctrl + d로 복사하여 프레임 사이의 거리는 자동으로 40px입니다.

2-6. 프레임 동시 선택
=> shift를 누르고 각 프레임을 선택하면 됩니다.
=> 프레임이 동시 선택 되었을 떄 프레임과 프레임 사이에 가운데에 bar가 존재합니다.
이를 좌-우로 드래그하면 프레임 간격을 조절할 수 있습니다.
=> 프레임이 동시 선택 되었을 떄 프레임 자체에 동그라미가 생깁니다.
이를 통해 프레임의 순서를 변경할 수 있습니다.

2-7. 프레임 선택
=> 프레임 내부에 여러 요소가 존재할 떄는 프레임 선택이 어렵습니다.
=> 그 떄는 프레임 이름을 선택하면 프레임이 선택됩니다.

2-8. 프레임 내에 엘리먼트 선택
=> ctrl 누르고 엘리먼트를 선택하면 됩니다.


3. ratangle, ellipse
3-1. R를 클릭하거나 하단메뉴에 Rectangle를 선택해서 드래그하여 사용합니다.
3-2. alt를 누르고 사이즈를 조절하면 가운데 맞춤으로 커집니다.
3-3. shift 누르고 사이즈를 조절하면 정사각형으로 커집니다.
3-4. alt + shift 사이즈 조절하면 가운데 + 정사각형

3-5. ratangle를 생성하면 모서리 밑 부분에 동그라미가 존재합니다.
=> 동그라미를 컨트롤하여 radius를 조절할 수 있습니다.

3-6. o를 클릭하면 원형을 그릴 수 있습니다.
3-7. alt, shift ractangle처럼 동작합니다.
3-8. 원형이 생성되면 동그라미 하나가 생성됩니다.
=> 부채꼴을 만들 수 있습니다.
3-9. 부채꼴이 완성되면 3개의 동그라미가 생깁니다.
=> 첫 번쨰 동그라미는 부채꼴, 두 번째는 ratio(원의 중간 파먹기), 세 번쨰는 원 돌리기


4. line, arrow
4-1. L이나 하단 메뉴 Line을 클릭하고 드래그하여 사용합니다.
4-2. alt를 누르면 가운데 맞춤으로 커집니다.
4-3. shift를 누르면 라인의 각도를 조절할 수 있습니다.

4-4. arrow는 shift + l이나 하단 메뉴 arrow를 클릭하고 드래그하여 사용합니다.
=> 4-2, 4-3과 동일합니다.


5. polygon (기본 모양은 삼각형)
=> 하단 메뉴에서 polygon를 선택하고 드래그하여 사용합니다.
=> polygon이 생성되면 동그라미가 두 개가 생성됩니다.
=> 하나는 radius, 다른 하나는 모서리를 증가시킵니다.


6. star (별 모양)
=> 하단 메뉴에서 star를 선택하고 드래그하여 사용합니다.
=> 동그라미 설정은 polygon과 동일하지만 ratio가 존재합니다.
